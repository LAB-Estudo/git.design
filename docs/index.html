Logo

The git logomark is comprised of a diamond and the word git. The diamod symbolizes how a distributed version control system works with the orange color being associated with git, a specific; a specific type of version control system. The text associated with the logomark is the name of the version control system.

Rationalization

The git logo is composed of a diamond with a vertical line and a diagnoal line. The vertical line symbolizes the master branch of development and the diagnoal line symobolizes a branch, one of the key features of a version control system. The orange diamond in the bakground represents the git itself.

The logo can be drawn on a 16px x 16px grid with the vertical bar and diagnoical bar each being 2px wide. The branch extends from master at a 45 degree angle to the edge of the diamond.
